## 枚举

```typescript
enum Direction {
  Up,
  Down,
  Left,
  Right
}

let direction: Direction = Direction.Up;
console.log(direction); // 输出: 0

direction = Direction.Right;
console.log(direction); // 输出: 3

// 通过枚举成员的值来访问枚举常量
let dirName: string = Direction[1];
console.log(dirName); // 输出: "Down"
```
